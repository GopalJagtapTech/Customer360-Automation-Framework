# Customer360 Automation Framework

Automation testing framework developed for Customer360 application using Selenium WebDriver, TestNG and Cucumber.

## Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Cucumber BDD
* Maven
* Extent Reports

## Framework Design

This framework follows Page Object Model (POM) design pattern to maintain reusable and scalable automation code.

## Key Features

✔ Page Object Model implementation
✔ Cucumber BDD feature files
✔ TestNG runner integration
✔ Screenshot capture on test failure
✔ Extent HTML report generation
